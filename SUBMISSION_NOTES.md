# Submission Notes

- Run `START_HERE.cmd` after extracting the ZIP on Windows 10/11.
- Supported launcher environment: Python 3.13.x.
- The Nikkei 225 page includes historical directional evaluation.
- The individual-stock page does **not** display or bundle individual-stock accuracy metrics. It presents the current six-level model output, 60-session and two-year charts, short- and medium/long-term analysis reports, factors, and feature importance.
- The take-profit / stop-loss simulator applies user-defined rules mechanically to historical OHLC data and does not use the forecast model.
